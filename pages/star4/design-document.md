# 页面4 — 埃尔林·哈兰德（Erling Haaland）主题展示页 设计文档

- **项目**：体育明星馆小组网页设计
- **负责页面**：页面4（star4）
- **开发分支**：`cyc`（不合并到主分支）
- **主题**：足球明星 —— 埃尔林·哈兰德（Erling Haaland）
- **展示内容**：成就荣誉、球队历程、成长经历、个人精彩操作

## 3. 配色方案

### 3.1 主色：曼城天蓝 `#6CABDD`

选择理由：哈兰德现效力于曼城，天蓝色是曼城最具代表性的品牌色，辨识度极高，且视觉上清新、活力、有运动感。

### 3.2 辅助色

| 颜色 | 色值 | 用途 |
|------|------|------|
| **深海军蓝** | `#1C2E4A` | 文字主色、深色背景、标题区域 |
| **白色** | `#FFFFFF` | 卡片背景、文字对比底色 |
| **金色** | `#D4A843` | 成就/荣誉装饰色、奖牌色、强调色 |
| **浅灰** | `#F0F4F8` | 页面背景、分割区域 |

### 3.3 配色逻辑
- 天蓝作为贯穿全页的主色调，用于 Hero 背景渐变、按钮、强调边框、图标
- 深海军蓝用于正文文字和深色区块，提供稳重感
- 金色仅用于荣誉成就相关元素，控制用量以提升珍贵感
- 浅灰作为内容区背景，与白色卡片形成层次

### 3.4 渐变方案（Hero 区域）
```
background: linear-gradient(135deg, #1C2E4A 0%, #6CABDD 50%, #D4A843 100%);
```
深海军蓝 → 天蓝 → 金色点缀，象征从青训到巅峰的历程。

## 4. 排版布局方案

### 4.1 布局策略：**纵向叙事流 + 时间线 + 网格混合布局**

采用"长滚动单页"结构，用户从上到下浏览，像阅读一个故事。每个区块采用不同的子布局避免单调：

| 区块 | 布局方式 |
|------|----------|
| Hero | 全屏宽幅，图片为背景或左右分栏（左文右图） |
| 基本信息 | 2 列网格卡片 |
| 成长经历 | 垂直时间线（左侧年份、右侧内容+图） |
| 球队历程 | 徽章网格（flex-wrap 自适应） |
| 成就荣誉 | 分类卡片网格（2 列），金色奖牌图标装饰 |
| 高光时刻 | 横向滚动卡片或 3 列等宽网格 |
| 数据统计 | 左右分栏（左标签右进度条） |
| 个人评价 | 全宽居中，引用样式 |

### 4.2 布局特点
- **Hero 区域**：全屏高度（100vh），图片作为背景覆盖，文字居中偏左叠加半透明遮罩
- **内容容器**：统一最大宽度 1120px，居中，与主站保持一致
- **时间线**：使用 CSS 伪元素（`::before` / `::after`）绘制垂直线和圆点，纯 CSS 实现
- **响应式**：平板端（≤900px）调整为 2 列，手机端（≤640px）全部单列，时间线转为左侧单边

## 6. 交互与动效设计

| 元素 | 动效 | 实现方式 |
|------|------|----------|
| Hero 标题 | 页面加载时从下方淡入上升 | CSS `@keyframes` + `animation-delay` |
| 时间线节点 | 滚动进入视口时从左/右滑入 | Intersection Observer API |
| 荣誉卡片 | 鼠标悬停上浮 + 阴影加深 | CSS `transition` |
| 徽章 | 鼠标悬停放大 + 金色描边 | CSS `transform: scale()` |
| 进度条 | 页面加载时从左到右填充动画 | CSS `@keyframes` + `width` 过渡 |
| 导航返回链接 | 悬停颜色过渡 | CSS `transition` |

## 7. 开发注意事项

1. **不修改其他页面**：仅修改 `pages/star4/` 下的三个文件
2. **不合并到主分支**：所有更改保留在 `cyc` 分支
3. **保持项目结构一致**：顶部保留"返回首页"链接，页脚与项目统一风格
4. **纯前端实现**：不引入后端或数据库，所有数据硬编码在 HTML/JS 中
5. **图片本地化**：建议在 `pages/star4/` 下创建 `images/` 目录存放所有图片素材
6. **无外部依赖**：除 Google Fonts 外，不引入外部库（如 Bootstrap、jQuery）
7. **语义化 HTML**：使用 `<section>`、`<article>`、`<figure>`、`<time>` 等标签
8. **无障碍**：图片加 `alt` 属性，颜色对比度符合 WCAG AA 标准
