# 体育运动明星主题展示网站项目介绍文档

## 一、项目基本信息

项目名称：体育运动明星主题展示网站

项目类型：网络应用开发课程项目 / 静态前端展示网站

项目主题：以体育明星为核心，展示不同运动项目中代表性人物的成长经历、职业成就、精神品质和高光时刻。

项目页面组成：

- 主页面：首页入口，负责统一展示四个运动主题并跳转到子页面。
- 子页面一：篮球主题，介绍科比·布莱恩特。
- 子页面二：乒乓球主题，介绍樊振东、马龙、张继科三位国乒代表人物。
- 子页面三：排球主题，介绍朱婷。
- 子页面四：足球主题，介绍埃尔林·哈兰德。

项目技术栈：

- HTML：负责网页内容结构。
- CSS：负责页面样式、布局、动画和响应式适配。
- JavaScript：负责页面交互效果，比如轮播图、按钮切换、滚动动画等。
- 图片资源：负责增强视觉表现，让人物介绍更直观。
- Google Fonts：部分子页面使用网络字体，让页面更有杂志感和设计感。

项目特点：

- 页面主题统一，都是围绕体育明星展开。
- 每个子页面有不同视觉风格，能体现不同运动项目的特色。
- 页面包含导航、图文介绍、数据展示、时间线、相册、互动切换等模块。
- 项目不依赖后端，直接打开 HTML 文件即可运行，适合课程展示和答辩演示。

## 二、项目整体结构

项目根目录结构如下：

```text
douxing/
├── index.html              # 主页面
├── README.md               # 项目说明
├── css/
│   └── main.css            # 主页面样式
├── js/
│   └── main.js             # 主页面轮播交互
├── picture/                # 主页面图片资源
└── pages/
    ├── zy/                 # 科比篮球主题页
    ├── star2/              # 国乒主题页
    ├── star3/              # 朱婷排球主题页
    └── star4/              # 哈兰德足球主题页
```

这个结构的好处是分工清晰：主页文件放在根目录，四个子页面分别放在 `pages` 文件夹下。每个子页面都有自己的 HTML、CSS、JS 和图片资源，方便小组成员独立开发，也方便后期维护。

## 三、主页面介绍

主页面文件：

- `index.html`
- `css/main.css`
- `js/main.js`

主页面定位：

主页面是整个网站的入口，相当于一个“体育明星主题展厅”。用户进入网站后，可以先看到四位/四组体育明星的整体展示，再通过轮播卡片或顶部导航进入对应子页面。

主页面主要内容：

1. 顶部导航栏

顶部导航栏包含网站名称和四个运动分类入口：

- 篮球
- 乒乓球
- 排球
- 足球

用户点击不同导航项，可以跳转到对应的子页面。

2. 首页主视觉文案

首页通过标题和简介说明项目主题：不同运动员虽然来自不同项目，但他们都有共同点，那就是通过训练、坚持和关键时刻的发挥，把天赋变成真正的竞技能力。

3. 人物轮播展示

主页中间是一个大型轮播区域，四张轮播图分别对应：

- 科比·布莱恩特
- 国乒三重火力
- 朱婷
- 埃尔林·哈兰德

每张轮播图都包含人物图片、运动类型、人物名称和一句简短介绍。用户可以点击轮播图进入对应页面。

4. 底部主题说明

主页下方通过引用和说明文字总结项目主题：这些体育明星不只是“赢得比赛”，更重要的是他们在压力、困难和挑战中塑造出了自己的精神品质。

主页面交互效果：

- 自动轮播：页面会每隔 4 秒自动切换到下一张人物卡片。
- 左右按钮：用户可以手动点击上一张或下一张。
- 指示点：底部圆点显示当前轮播位置，也可以点击切换。
- 平滑动画：轮播切换时使用 CSS `transform` 实现横向滑动。

答辩介绍示例：

“主页面是我们项目的统一入口，它通过导航栏和轮播图把四个子页面连接起来。轮播图不仅起到视觉展示作用，也承担页面跳转功能。我们用 JavaScript 控制当前显示的轮播项，用 CSS 的 transform 实现滑动动画，让首页看起来更完整、更有交互感。”

## 四、子页面一：科比篮球主题页

页面文件：

- `pages/zy/kobe.html`
- `pages/zy/kobe.css`
- `pages/zy/kobe.js`

页面主题：

该页面介绍篮球明星科比·布莱恩特，重点展示他的职业生涯、代表荣誉和“曼巴精神”。

页面主要内容：

1. 页面导航

页面顶部有固定导航，包含：

- 首页
- 职业生涯
- 荣誉殿堂
- 曼巴精神

用户可以通过导航快速跳转到页面不同区域。

2. 首页介绍区域

页面开头展示科比的主视觉图片、姓名、生卒年份和人物简介。简介部分说明科比效力洛杉矶湖人 20 个赛季，是篮球史上重要的得分手之一。

3. 数据统计区域

页面展示科比的关键数据，例如：

- 生涯总得分
- 出战场次
- 总冠军数量

这些数字不是直接静态显示，而是通过 JavaScript 做了数字递增动画，页面打开时会从 0 逐步增长到目标数值。

4. 职业生涯时间线

页面用时间线展示科比的关键生涯节点：

- 1996 年进入 NBA
- 2000-2002 年湖人三连冠
- 2006 年单场 81 分
- 2008-2010 年再次达到职业巅峰
- 2016 年退役战砍下 60 分

时间线能让观众按照年份理解人物经历，比单纯文字介绍更清晰。

5. 荣誉殿堂

该区域使用卡片展示科比的重要荣誉，包括 NBA 总冠军、总决赛 MVP、常规赛 MVP、全明星、奥运金牌等。

6. 曼巴精神名言轮播

页面底部设计了名言轮播模块，用户可以点击按钮切换不同名言，也可以等待自动切换。

该页面用到的技术亮点：

- `DOMContentLoaded`：确保网页内容加载完再执行 JavaScript。
- `requestAnimationFrame`：实现数字增长动画，让数据展示更有动感。
- `IntersectionObserver`：检测时间线元素是否进入屏幕，进入后添加动画类名。
- `scroll` 事件监听：根据滚动位置高亮当前导航项。
- `setInterval`：实现名言自动轮播。

小白解释：

可以把 HTML 理解成“页面骨架”，CSS 是“衣服和排版”，JavaScript 是“让页面动起来的大脑”。科比页面里的数字动画、滚动出现、名言切换，都是 JavaScript 在页面加载后控制的。

答辩介绍示例：

“科比页面重点体现人物传奇感，所以我们设计了大图开场、职业时间线和荣誉卡片。技术上使用 JavaScript 实现数字递增、滚动出现和名言轮播，让页面不只是静态介绍，而是有浏览节奏和互动效果。”

## 五、子页面二：国乒主题页

页面文件：

- `pages/star2/index.html`
- `pages/star2/style.css`
- `pages/star2/script.js`

页面主题：

该页面以“国乒三重火力”为主题，集中展示樊振东、马龙、张继科三位中国男子乒乓球代表人物。整体风格接近杂志封面和体育海报。

页面主要内容：

1. 杂志封面式首屏

页面首屏使用大标题、人物拼贴图、关键词和版面元素，营造“体育杂志专题”的感觉。

2. 封面故事

该部分解释为什么把三位运动员放在同一页面：他们分别代表力量相持、完整控制和大赛爆发三种不同气质。

3. 三位主角切换

页面设置了三个按钮：

- 樊振东
- 马龙
- 张继科

用户点击不同按钮时，对应人物介绍卡片会切换显示。这个功能让页面在有限空间里展示更多内容。

4. 技术火力区域

该区域从乒乓球比赛角度介绍技术能力，例如：

- 发接发
- 正反手衔接
- 相持抗压
- 关键分爆发

页面还使用进度条样式强化视觉表达。

5. 高光回合与海报相册

页面后半部分通过图文卡片和瀑布流相册展示运动员的代表瞬间，增强画面感。

该页面用到的技术亮点：

- CSS Grid：用于构建复杂的杂志式布局。
- CSS Flex：用于导航、标签、按钮等横向排列。
- `data-*` 自定义属性：用来标记当前点击的是哪位球员。
- JavaScript 点击事件：点击球员按钮时切换对应人物卡片。
- `IntersectionObserver`：滚动到某个区域时触发出现动画。
- CSS 变量：通过 `--stagger-index` 控制子元素依次出现的动画延迟。

小白解释：

`data-*` 可以理解为给 HTML 元素贴标签。比如按钮上写着 `data-player="fzd"`，JavaScript 就知道这个按钮代表樊振东。点击按钮后，JS 找到同样标签的人物卡片，把它显示出来，把其他卡片隐藏。

答辩介绍示例：

“国乒页面的重点是视觉设计和人物切换。我们没有把三位运动员简单平铺，而是做成杂志专题形式。点击不同人物按钮时，JavaScript 会读取按钮上的 data-player 属性，再切换对应的人物介绍卡片，这样页面更节省空间，也更有互动性。”

## 八、项目使用技术详解

### 1. HTML：搭建网页骨架

HTML 的作用是定义网页里有什么内容。

在本项目中，HTML 负责：

- 写出标题、段落、图片、按钮和链接。
- 划分页面区域，例如导航栏、主视觉、人物介绍、荣誉展示、相册等。
- 设置页面之间的跳转链接。
- 给元素添加类名，方便 CSS 和 JavaScript 找到它们。

小白理解：

HTML 就像房子的毛坯结构。它决定哪里是门、哪里是窗、哪里是客厅、哪里是卧室。没有 HTML，网页就没有内容和结构。

项目例子：

- `index.html` 中的 `<a>` 标签用于跳转到四个子页面。
- 各子页面的 `<section>` 标签用于划分不同内容模块。
- `<img>` 标签用于展示运动员图片。

### 2. CSS：控制页面外观

CSS 的作用是让网页变得好看。

在本项目中，CSS 负责：

- 设置字体、颜色、背景、间距。
- 控制图片大小和裁剪方式。
- 实现导航栏固定、卡片样式、阴影、圆角。
- 使用 Grid 和 Flex 完成页面排版。
- 实现轮播滑动、淡入、上移、缩放等动画。
- 使用媒体查询适配手机和电脑屏幕。

小白理解：

CSS 就像给房子装修。HTML 只是有墙和房间，CSS 决定墙是什么颜色、家具怎么摆、灯光是什么风格。

项目例子：

- 主页面用 CSS 把轮播图做成大幅视觉卡片。
- 国乒页面用 CSS 做出杂志封面和海报拼贴感。
- 哈兰德页面用 CSS 做出球场线条和入场动画。

### 3. JavaScript：实现页面交互

JavaScript 的作用是让网页能响应用户操作。

在本项目中，JavaScript 负责：

- 主页面轮播图自动切换。
- 点击上一张、下一张按钮切换轮播内容。
- 科比页面数字统计动画。
- 科比页面名言轮播。
- 国乒页面点击按钮切换人物介绍。
- 滚动到某个区域时触发内容出现动画。

小白理解：

JavaScript 就像网页的大脑和遥控器。HTML 决定有什么，CSS 决定长什么样，JavaScript 决定它能不能动、能不能点、能不能自动变化。

项目例子：

- `setInterval` 可以每隔几秒执行一次函数，所以能做自动轮播。
- `addEventListener` 可以监听用户点击按钮，所以能做点击切换。
- `IntersectionObserver` 可以判断内容是否进入屏幕，所以能做滚动出现动画。

### 4. CSS Grid：二维布局工具

CSS Grid 主要用于同时控制“行”和“列”的布局。

项目中 Grid 的作用：

- 做成就卡片网格。
- 做图片相册布局。
- 做人物介绍双栏布局。
- 做复杂的海报式页面结构。

小白理解：

Grid 就像表格或者棋盘，可以把页面分成很多格子，然后把图片、文字、卡片放到不同格子里。

### 5. CSS Flex：一维布局工具

Flex 主要用于横向或纵向排列元素。

项目中 Flex 的作用：

- 顶部导航横向排列。
- 标签和按钮横向排列。
- 卡片内部内容对齐。
- 让元素在不同屏幕宽度下自动换行。

小白理解：

Flex 就像一排可以自动调整位置的队伍。屏幕宽时排成一行，空间不够时可以换行。

### 6. 响应式设计：适配不同屏幕

响应式设计就是让网页在电脑、平板、手机上都能正常显示。

项目中响应式设计的作用：

- 手机屏幕下导航栏会换行或竖排。
- 大图和文字会自动缩小。
- 多列布局在小屏幕下会变成单列。
- 按钮和内容不会严重挤压。

小白理解：

响应式设计就像衣服有弹性，可以适应不同身材。网页也要适应不同屏幕大小。

### 7. IntersectionObserver：滚动触发动画

`IntersectionObserver` 是 JavaScript 提供的一个浏览器功能，用来观察某个元素是否进入屏幕。

项目中它的作用：

- 用户滚动页面时，检测某个区域是否出现。
- 出现后给元素添加 `is-visible` 或 `visible` 类名。
- CSS 根据这个类名播放淡入、上移动画。

小白理解：

它就像一个“观察员”，一直看着页面内容有没有进入用户视线。一旦看见了，就通知 CSS：“可以开始动画了。”

### 8. setInterval：定时执行

`setInterval` 是 JavaScript 的定时器。

项目中它的作用：

- 主页面每 4 秒自动切换轮播图。
- 科比页面每 5 秒自动切换名言。

小白理解：

它就像闹钟，每隔固定时间提醒网页执行一次动作。

### 9. addEventListener：监听用户操作

`addEventListener` 用来监听用户行为。

项目中它的作用：

- 监听轮播按钮点击。
- 监听国乒人物按钮点击。
- 监听页面滚动。
- 监听名言切换按钮点击。

小白理解：

它就像一个按钮旁边的工作人员，用户一按按钮，它就通知网页执行对应动作。

### 10. 图片资源：增强视觉表达

图片是本项目很重要的部分。

项目中图片的作用：

- 帮助用户快速识别人物。
- 增强体育主题氛围。
- 让页面从普通文字介绍变成视觉化展示。
- 适合答辩时投屏展示。

小白理解：

如果只有文字，页面会像资料表；加入图片后，页面更像一个完整的主题展览。

各位老师好，我们小组的项目是“体育运动明星主题展示网站”。这个项目是一个静态前端网站，主要使用 HTML、CSS 和 JavaScript 完成。网站以体育明星为主题，包括一个主页面和四个子页面，分别展示篮球、乒乓球、排球和足球项目中的代表人物。

首先是主页面。主页面是整个项目的入口，上方有导航栏，中间是人物轮播图。轮播图展示四个主题页面，用户可以点击图片或导航进入对应页面。这个轮播功能是用 JavaScript 控制当前显示的页面，再用 CSS 的 transform 实现滑动效果。

第一个子页面是科比页面。这个页面主要展示科比的职业生涯、荣誉和曼巴精神。页面中有数字增长动画、职业时间线、荣誉卡片和名言轮播。数字增长用到了 JavaScript 的 requestAnimationFrame，时间线出现效果用到了 IntersectionObserver。

第三个子页面是朱婷页面。这个页面更偏人物档案式介绍，重点展示朱婷的个人资料、代表成就、高光时刻和人物精神。这个页面主要使用 HTML 组织内容，用 CSS Grid 和 Flex 完成排版。

第四个子页面是哈兰德页面。这个页面使用足球场线条、大号 09 数字和图片营造足球氛围。页面中有数据卡片、职业章节、高光相册、生涯路线和荣誉墙。滚动动画同样使用 IntersectionObserver 触发。

从技术上看，HTML 负责页面内容结构，CSS 负责视觉样式和响应式布局，JavaScript 负责轮播、点击切换、数字动画和滚动动画。通过这个项目，我们练习了网页结构设计、页面布局、图片展示、动画效果和基础交互功能，也提升了小组分工协作能力。

## 十三、可能被老师问到的问题

### 1. 这个项目有没有使用后端？

没有。这个项目是静态前端项目，主要使用 HTML、CSS 和 JavaScript 完成，不涉及后端服务器和数据库。

### 2. 为什么要做主页面和四个子页面？

主页面用于统一展示项目主题和页面入口，四个子页面分别展示不同运动项目和人物。这样结构更清晰，也方便小组成员分工。

### 3. JavaScript 在项目中主要做了什么？

主要做交互效果，包括首页轮播、按钮切换、数字动画、名言轮播和滚动出现动画。

### 4. CSS Grid 和 Flex 有什么区别？

Grid 更适合做多行多列的整体布局，比如卡片网格、相册布局；Flex 更适合做一行或一列的排列，比如导航栏、按钮组、标签组。

### 5. 什么是响应式设计？

响应式设计就是让网页在不同屏幕上都能正常显示。比如电脑上可以多列展示，手机上可以自动变成单列展示。

### 6. IntersectionObserver 是干什么的？

它用于检测页面元素是否进入屏幕。当用户滚动到某个区域时，JavaScript 给元素添加类名，CSS 再播放动画。

### 7. 这个项目最大的亮点是什么？

项目不仅完成了基本人物介绍，还加入了轮播、切换、动画、时间线、相册等展示方式，让页面更像一个完整的体育明星主题展览。
